Программа ProjectManager.jar предназначена для сотрудников компаний, которые занимаются разработкой проектов. В этой программе сотрудник ведёт учёт проектов, может создавать отчёты по интересующей его информации: отчет о статусе проекта ,отчет о рисках и задачах проекта.
